FluxDown 主窗口采用三栏布局:左侧可调整宽度的**侧边栏**、居中的**任务列表**,以及选中任务后才出现的**详情面板**。

## 布局概览

- **侧边栏** —— 拖拽右边缘可在 180px–320px 之间调整宽度。若关闭了全部侧边栏区块(见下文),侧边栏会整体隐藏,任务列表占满剩余宽度。
- **任务列表** —— 始终占据侧边栏与详情面板之间的剩余空间。
- **详情面板** —— 点击任务行前不显示;拖拽左边缘可在 240px–420px 之间调整宽度。再次点击同一任务行,或点击面板的关闭按钮即可收起。

## 侧边栏

三个可独立开关的可折叠区块:

- **状态** —— 全部/下载中/已完成/已暂停/失败,各自显示实时任务数;"下载中"一项在有任务传输时还会显示一个活动小圆点。点击某一项即可按其筛选任务列表。
- **队列**(默认展开)—— 内置的**主队列**与**稍后下载**队列,加上你创建的所有命名队列(点击 **+** 新建)。每个队列前有运行/停止状态圆点;悬浮条目可见启停、管理、删除(仅自定义队列)按钮,右键菜单提供同样的操作。队列管理对话框有三个 Tab——**设置**(名称、速度限制、最大并发、保存目录、默认线程数、默认 User-Agent)、**定时**(每日启停时刻与生效星期;启用定时的队列显示小闹钟图标)、**任务顺序**(队列启动时任务的恢复顺序)。队列启停机制详见[快速上手](/docs/zh/getting-started/quickstart/#启动与停止队列)。
- **分类**(默认折叠)—— 7 个内置文件类型筛选(全部文件、视频、音频、文档、图片、压缩包、其他),外加你在**设置 → 通用 → 自定义分类**中按扩展名或正则表达式定义的自定义分类。

右键区块标题可选择**隐藏此区块**,之后可在**设置 → 通用 → 侧边栏显示**中重新开启。侧边栏底部显示当前版本号,有新版本时会出现内联更新按钮。

## 顶栏

- **新建下载** —— 打开新建下载对话框。
- **搜索框**(<kbd>Ctrl</kbd>/<kbd>Cmd</kbd>+<kbd>F</kbd>)—— 边输入边同时搜索任务名/URL 与设置项,方向键在结果间移动,回车跳转到选中结果。
- **工具按钮**(右上角,窗口控制按钮旁)—— 全部暂停、全部恢复、设置、亮暗主题切换。每个按钮都可在**设置 → 通用 → 标题栏按钮**中单独隐藏,也可直接右键该按钮隐藏。

macOS 上红黄绿交通灯窗口按钮位于左上角;Windows/Linux 上工具按钮与窗口控制按钮一起分组在右上角。

## 任务列表

列头依次为文件名/进度/速度/剩余时间/状态,文件名列旁还有一个管理模式切换按钮。

- **管理模式** —— 按 <kbd>Ctrl</kbd>/<kbd>Cmd</kbd>+<kbd>A</kbd> 或点击切换按钮进入。列头变为全选复选框,顶部区域变为批量操作栏(全选/取消全选、已选计数、删除任务、删除任务和文件、取消)。
- **分组** —— 顶部是始终展开的**"正在下载"**活跃组,汇总下载中/排队中/准备中的任务(自带一键全部暂停),其后是按任务创建时间划分、可折叠的时间分组——今天、昨天、最近一周、最近一个月、更久以前。点击分组标题可折叠/展开。
- **双击**任务行:进行中的任务暂停,已暂停/失败的任务恢复,已完成的任务打开文件。
- **右键**任务行弹出完整菜单,自上而下:暂停/继续、优先下载/取消优先、打开文件(仅已完成任务)/打开所在文件夹、复制下载地址、删除任务、删除任务和文件。
- **右键列表下方空白处**弹出新建下载、全部开始、全部暂停。

## 详情面板

点击任意任务打开面板,自上而下为:

- **文件信息** —— 图标、扩展名、完整文件名。
- **进度** —— 大号百分比读数、分段进度条,以及(有分段数据时)IDM 风格的像素网格——每个格子反映对应字节区间的下载填充状态,下方附带列出各分段编号与百分比的图例。当 FluxDown 主动或响应式地拆分慢速分段时,进度区域还会显示累计拆分次数及最近一次拆分的字节范围。
- **信息表** —— 大小、已下载、速度、剩余(ETA)、状态、线程数(仅进行中显示)、路径、URL(带一键复制按钮),任务失败时还会显示错误信息行。
- **操作按钮** —— 暂停/继续按钮,以及一个危险操作样式的"删除任务和文件"按钮。

## 状态栏

窗口底部的细条状态栏:

- **左侧** —— 状态圆点(有任务活跃时为绿色,空闲时为灰色)配文字说明、实时总下载速度,以及"N 活跃 · N 暂停 · N 总计"的汇总文字。
- **右侧** —— 限速弹出面板(开关 + 预设值 128 KB/s / 512 KB/s / 1 MB/s / 2 MB/s / 5 MB/s,或自定义数值)、完成后关机弹出面板(预设 0/1/5/10/30 分钟,启用后显示实时倒计时),以及打开反馈对话框的**反馈**按钮。

## 全局快捷键

以下快捷键在主窗口任意位置均可使用(设置页或任意对话框打开时不响应):

| 快捷键 | 作用 |
|---|---|
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd>+<kbd>F</kbd> | 聚焦搜索框 |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd>+<kbd>A</kbd> | 全选当前可见任务(自动进入管理模式) |
| <kbd>Esc</kbd> | 退出管理模式 |
| <kbd>Delete</kbd>(macOS 上也可用 <kbd>Cmd</kbd>+<kbd>Backspace</kbd>) | 删除已勾选的任务,会弹出确认提示(仅管理模式下生效) |

## 系统托盘

窗口最小化或关闭到托盘后,FluxDown 会保留一个托盘图标。右键点击可看到:

- **显示主窗口** —— 将主窗口重新置于前台。
- **显示悬浮球**(复选框)—— 一个桌面常驻置顶小组件,显示实时速度与进度,可直接把链接或种子文件拖到它上面创建下载任务。在 Wayland 环境下不可用,此时托盘会改为显示实时速度,复制的链接会在你重新打开主窗口时自动填入下载对话框。
- **退出** —— 退出 FluxDown。
